Vídeos Rápidos

Bem-vindo ao aplicativo Vídeos Rápidos. Aqui você encontra nossa Política de Privacidade e Termos de Uso.

Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo Vídeos Rápidos coleta, usa e protege as informações dos usuários.

1. Informações Coletadas

O aplicativo não coleta dados pessoais sensíveis como CPF, endereço ou dados bancários.

Podem ser coletadas informações técnicas, como modelo do dispositivo, sistema operacional, idioma e dados de uso básicos (erros e desempenho), apenas para melhorar o funcionamento do app.

Os textos digitados para geração de vídeos (“prompts”) podem ser enviados a serviços de terceiros de inteligência artificial, exclusivamente para gerar o conteúdo solicitado pelo usuário.

2. Serviços de Terceiros

Os prompts são enviados ao serviço de geração de IA Luma AI para que o vídeo seja criado.

3. Armazenamento

Os vídeos gerados podem ser armazenados temporariamente em servidores da IA para disponibilização ao usuário.

4. Conteúdo Gerado pelo Usuário

O usuário é responsável pelo conteúdo que gerar e deve respeitar direitos autorais, de imagem e leis aplicáveis.

5. Idade Recomendada

O aplicativo é recomendado para maiores de 13 anos.

Termos de Uso

Ao utilizar o aplicativo Vídeos Rápidos, você concorda com estes Termos.

1. Responsabilidade do Usuário

O usuário não deve gerar vídeos ofensivos, ilegais ou que infrinjam direitos autorais.

2. Limitação de Responsabilidade

O desenvolvedor não se responsabiliza pelo conteúdo criado pelo usuário ou por danos causados pelo uso do aplicativo.
